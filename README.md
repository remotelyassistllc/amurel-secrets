# Amurel Secrets 🖤✨
### Luxury African Perfume Store

A premium e-commerce perfume store built on Base44, targeting the African market.

**Live site:** https://amurelsecrets.com

## Brand Identity
- **Colors:** Deep Gold (#C9A84C), Rich Black (#0a0a0a), Cream White (#F5F0E8)
- **Aesthetic:** Dark luxury
- **Market:** Africa (mobile-first)

## Tech Stack
- React + Tailwind CSS
- Base44 platform (entities, auth, hosting)
- Domain: amurelsecrets.com (via Cloudflare DNS)

## Entities
- `Product` — Perfume catalog with pricing, notes, ratings
- `Order` — Customer orders & fulfillment
- `Review` — Product reviews
- `Wishlist` — User wishlists

## Development
Built and managed via [Base44](https://base44.com).
